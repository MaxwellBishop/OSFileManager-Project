# OSFileManager-Project

This simulation represents how the OS manages the long-term storage and the logical-to-physical mapping of data. To that end, this group will build a User-Space 
Virtual File System (VFS) that includes:

Creating a File Allocation Table (FAT) or Indexed Allocation (Inodes).
Supporting basic CLI commands: create, delete, open, read, and write.
Implementing a Directory Tree structure within a single large binary file (the "virtual disk").
